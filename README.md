# ⚾ 미션 - 숫자 야구 게임


## 🎯 기능 요구사항

- 1~9까지 뽑을수 있는 랜덤함수
- 배열중 제시한 숫자가 맞는지 확인하는 함수
  - 스트라이크
  - 볼
  - 낫싱
- 이 같은 과정을 반복해 컴퓨터가 선택한 3개의 숫자를 모두 맞히면 게임이 종료된다.
- 게임을 종료한 후 게임을 다시 시작할 수 있다.
- 게임을 종료한 후 id가 `game-restart-button`인 버튼을 클릭함으로써 게임을 다시 시작할 수 있다. 
  - `예) <button id="game-restart-button">재시작</button>`
